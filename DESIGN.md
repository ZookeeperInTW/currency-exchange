# 設計規範（北歐簡約風）

本文件為專案唯一設計依據，所有 UI 實作需依此文件的色票、字型、間距與元件規範，不自行發明樣式。

## 色票

北歐風格採溫暖中性底色、低飽和度主色，以及大地色系的語意色（避免高彩度的螢光色），維持簡約排版但整體色調更柔和自然。

| 用途 | 色碼 | 說明 |
|------|------|------|
| 背景 | `#FAF8F4` | 頁面主背景（暖白，非純白） |
| 次背景 / 卡片背景 | `#F1ECE3` | 卡片、輸入框底色（暖灰米色） |
| 主要文字 | `#2B2A28` | 標題、數值（暖炭灰，非純黑） |
| 次要文字 | `#7C766C` | 說明文字、標籤 |
| 邊框 | `#E2DCCF` | 卡片邊框、分隔線 |
| 主色（強調色） | `#5B7C8D` | 主要按鈕、聚焦狀態、連結、圖表主線（霧藍） |
| 主色（hover） | `#486575` | 主色 hover 狀態 |
| 上漲 / 正向 | `#7C9473` | 趨勢圖漲幅（鼠尾草綠） |
| 下跌 / 負向 | `#BC6A4C` | 趨勢圖跌幅、錯誤訊息（陶土橘） |

## 字型

- 字體：`Inter`（透過 `next/font/google` 載入），fallback 為系統字體 `-apple-system, "Segoe UI", sans-serif`
- 標題（H1）：32px / 700 / line-height 1.25
- 卡片標題（H2）：18px / 600
- 轉換結果數值：40px / 700（強調用大字級展示金額）
- 換算總額（卡片內最終加總金額）：56px / 800 / 主色 `#5B7C8D`（比一般轉換結果數值更強調）
- 內文：16px / 400
- 說明文字（caption）：13px / 400 / 顏色使用次要文字色

## 間距（8px 基準網格）

`4 / 8 / 12 / 16 / 24 / 32 / 48 / 64` px，元件內距與元件間距一律取用此數列。

## 元件規範

- **卡片（Card）**：背景 `#F1ECE3`、邊框 `1px solid #E2DCCF`、圓角 `12px`、內距 `24px`
- **輸入框（Input）**：白底、邊框 `#E2DCCF`、圓角 `8px`、內距 `12px 16px`、focus 時邊框轉為主色 `#5B7C8D` 並加 2px 主色 20% 透明度外框
- **按鈕（Primary Button）**：主色底、白字、圓角 `8px`、內距 `12px 24px`、hover 轉 `#486575`
- **幣別切換 Tab**：底線式 tab，選中時底線與文字為主色，未選中為次要文字色
- **趨勢圖（Chart）**：單線圖，線條使用主色 `#5B7C8D`，區域填色為主色 8% 透明度，格線使用邊框色且極淡

## 網站 Icon

- **Favicon**：32×32 圓角矩形（圓角 7px），底色為主色 `#5B7C8D`，置中白色粗體 `$` 符號（字級 21px / 800），以 `app/icon.tsx`（`next/og` `ImageResponse`）動態產生
- **Apple Touch Icon**（加入 iPhone 主畫面用）：180×180 滿版正方形（不加圓角，交由 iOS 自行套用遮罩），底色為主色 `#5B7C8D`，置中白色粗體 `$` 符號（字級 108px / 800），以 `app/apple-icon.tsx` 動態產生
- 主畫面顯示名稱：透過 `appleWebApp.title` 設為「匯率換算器」，避免使用完整頁面標題被截斷
- 皆不使用外部圖檔

## 版面

- 單欄置中版面，最大寬度 `720px`
- 頁首：網站標題 + 一句話說明
- 主卡片：即時換算台幣總額（置頂強調）+ 幣別輸入區（三種幣別可分別輸入，位於總額下方）
- 次卡片：匯率趨勢圖（可切換 USD / JPY / PHP）
- 頁尾：資料來源標註（台灣銀行牌告匯率，透過 FinMind 提供，含資料日期）

## 響應式

- 手機（< 640px）：卡片內距縮為 `16px`，數值字級縮為 `32px`，換算總額字級縮為 `40px`
- 桌面（≥ 640px）：採用上述基準規格
